# 🎮 除法對戰遊戲

國小除法計算多人即時對戰教學遊戲

## 功能特色
- 多人即時對戰（WebSocket）
- 老師建立房間 → 學生輸入代碼加入
- 2~3位數 ÷ 1位數，含餘數
- 答題速度計分 + 幸運雙倍機制
- 老師即時監控每位學生作答狀況
- 完整成績報表

## 🚀 Zeabur 部署步驟

### 前置準備
1. 確認你有 [GitHub](https://github.com) 帳號
2. 確認你有 [Zeabur](https://zeabur.com) 帳號

### 步驟一：上傳到 GitHub

1. 登入 GitHub，點右上角 **+** → **New repository**
2. Repository name 填：`division-game`
3. 選 **Public**，點 **Create repository**
4. 依照 GitHub 頁面上的指示，上傳這個資料夾的所有檔案

### 步驟二：在 Zeabur 部署

1. 登入 [zeabur.com](https://zeabur.com)
2. 點擊 **New Project**
3. 點擊 **Add Service** → **Deploy from GitHub**
4. 授權連接 GitHub，選擇 `division-game` 這個 repository
5. Zeabur 會自動偵測並部署

### 步驟三：取得網址

- 部署完成後，點 **Domains** → **Generate Domain**
- 你會得到一個 `xxx.zeabur.app` 的網址
- 這就是遊戲網址！

## 🎯 遊戲操作說明

### 老師操作
1. 打開遊戲網址，點 **老師入口**
2. 點 **建立新房間**，記下6碼房間代碼
3. 告訴學生這個代碼
4. 等學生都加入後，點 **開始遊戲**
5. 遊戲進行中可即時看到每位學生的作答狀況
6. 15題結束後可查看完整成績報表

### 學生操作
1. 打開遊戲網址，點 **學生入口**
2. 輸入老師給的6碼房間代碼和自己的名字
3. 等待老師開始遊戲
4. 每題15秒，填入**商**和**餘數**（沒有餘數填0）
5. 點**送出答案**

### 計分規則
| 情況 | 分數 |
|------|------|
| ≤3秒答對 | +100分 |
| 3~6秒答對 | +80分 |
| 6~10秒答對 | +60分 |
| 10~15秒答對 | +40分 |
| 答錯 | -20分（最低0分）|
| 答對 >5秒（20%機率）| ⭐ 雙倍分數！|
| 未作答 | 0分 |
